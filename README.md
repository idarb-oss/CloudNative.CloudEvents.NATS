## About The Project

CloudNative.CloudEvents.NATS adds extension methods on top of CloudEvent and NATS Msg classes to easily convert between them.

<p align="right">(<a href="#top">back to top</a>)</p>



### Built With

* [.NET](https://dotnet.microsoft.com/)
